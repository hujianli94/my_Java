12.7. 抽象类与接口区别¶

归纳抽象类与接口区别如下：

  1. 接口支持多继承，而抽象类（包括具体类）只能继承一个父类。
  2. 接口中不能有实例成员变量，接口所声明的成员变量全部是静态常量，即便是变量不加public static final修饰符也是静态常量。抽象类与普通类一样各种形式的成员变量都可以声明。
  3. 接口中没有包含构造方法，由于没有实例成员变量，也就不需要构造方法了。抽象类中可以有 实例成员变量，也需要构造方法。
  4. 抽象类中可以声明抽象方法和具体方法。Java 8之前接口中只有抽象方法，而Java 8之后接口中 也可以声明具体方法，具体方法通过声明默认方法实现。

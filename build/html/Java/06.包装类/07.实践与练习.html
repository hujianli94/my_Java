6.7. 实践与练习¶

6.7.1. 练习1¶

  • 创建Integer类对象，并以int类型将Integer的值返回
public class GetInt {
    public static void main(String[] args) {
        Integer inte = new Integer(456);
        int myint = inte.intValue();
        System.out.println("Interger inte 转为int的值为 " + myint);
    }

}

/*
Interger inte 转为int的值为 456
 */

6.7.2. 练习2¶

创建两个Character对象，通过equals()方法比较它们是否相等。

public class GetInt {
    public static void main(String[] args) {
        Character mychar1 = new Character('H'); // 声明Character对象1
        Character mychar2 = new Character('h'); // 声明Character对象2
        Character mychar3 = Character.toLowerCase(mychar1);
        Character mychar4 = Character.toLowerCase(mychar2);

        if (mychar3.equals(mychar4)) {
            System.out.println(mychar3 + "等于"+mychar4);
        } else {
            System.out.println(mychar3 + "不等于"+mychar4);

        }


}
}
/*
h等于h
 */

练习3¶

编写程序，实现通过字符型变量创建boolean值，再将其转换成字符串，观察输出后的字符串与 创建Boolean对象时给定的参数是否相同。

public class CreateBoolean {
    public static void main(String args[]) {
        Boolean b = new Boolean("ok");
        String str = b.toString();
        System.out.println("ok:" + str);
        b = new Boolean("true");
        str = b.toString();
        System.out.println("true:" + str);
    }
}

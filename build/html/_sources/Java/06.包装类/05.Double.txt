Double
======

Double和Float包装类是对double、float基本类型的封装，它们都是Number的子类，
又都是对小数进行操作，所有常用方法基本相同。

该类提供多个方法，可以将double转换为String，也可以将String转换为double。

|image0|

常量
----

::

   //创建类
   public class Getboolean1 {
   //  创建主方法
   public static void main(String[] args) {
       Double my_double = new Double(3.13);
   //  返回int,表示有限double变量可能具有的最大指数
       System.out.println(Double.MAX_EXPONENT);
   //  返回int，表示标准化double变量可能具有的最小指数
       System.out.println(Double.MIN_EXPONENT);
   //  返回double值，表示保存double类型的负无穷大值的常量
       System.out.println(Double.NEGATIVE_INFINITY);
   //  返回double值，表示保存double类型的正无穷大值的常量
       System.out.println(Double.POSITIVE_INFINITY);
       System.out.println(my_double);

   }
   }

   /*
   输出内容
   1023
   -1022
   -Infinity
   Infinity
   3.13
   */

Character
=========

Character类在对象保重一个基本类型为char的值，
一个Character类型的对象包含
类型为char的单个字段，该类提供了几个方法，主要是将字符从大写转为小写，反之。

eg

::


   public class Character_str { 
       public static void main(String args[]) { // 主方法
           Character mychar1 = new Character('A'); // 声明Character对象
           Character mychar2 = new Character('a'); // 声明Character对象
   //      判断字符串是否为大写
           System.out.println(mychar1 + "是大写字母吗? "+ Character.isUpperCase(mychar1));
   //      将字符串转换为大写
           System.out.println("将对象" + mychar1 + "转为小写后的结果为："+Character.toLowerCase(mychar1));
   //      判断字符串是否为小写
           System.out.println(mychar2 + "是小写字母吗? "+ Character.isLowerCase(mychar2));
   //      将字符串转换为小写
           System.out.println("将对象" + mychar1 + "转为大写后的结果为："+Character.toUpperCase(mychar1));
   //      返回此Character的值
           System.out.println("返回此Character的值为："+ Character.toString(mychar1));
   //      比较两个对象是否相同
           System.out.println(mychar1.equals(mychar2));
           
       }
   }

   /*
   A是大写字母吗? true
   将对象A转为小写后的结果为：a
   A是小写字母吗? false
   将对象A转为大写后的结果为：A
   返回此Character的值为：A
   true

    */
   */
